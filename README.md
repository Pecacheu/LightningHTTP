# LightningHTTP
High performance minimal C++ HTTP Server w/ HTTPS Support

Requirements:
- C++14 & std library (including std::thread)
- C++ utils.h & net.h
- OpenSSL 1.1+
- Optional: Linux sys/utsname.h
